# 🏠 House Service

🌐 **Demonstração Online:**  
[Landing Page](https://naclaragsd.github.io/houseService/)

## 📖 Sobre o Projeto

O House Service é uma plataforma web criada para facilitar a conexão entre clientes e profissionais de serviços domésticos. O projeto apresenta uma interface moderna, intuitiva e responsiva, permitindo a navegação por diferentes seções e a visualização dos serviços oferecidos.

## ✨ Funcionalidades

- Navegação por seções da página
- Layout responsivo para diferentes dispositivos
- Barra de navegação com efeito ao rolar a página
- Animações de entrada dos elementos durante a navegação
- Interface moderna e organizada
- Experiência de usuário otimizada

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Desenvolver uma aplicação web de landing page para praticar conceitos fundamentais de desenvolvimento front-end, incluindo estruturação de páginas, estilização responsiva e interatividade com JavaScript. O projeto foi realizado como parte das atividades acadêmicas da disciplina de Desenvolvimento Front-End.
